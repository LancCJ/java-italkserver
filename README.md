# 项目简介

    主要用于项目react-native-italk该App后台服务及管理

# 相关技术

    1.Jfianl
    2.Maven
    3.AmazeUI
    
# 使用步骤
    
    1.Clone到本地
    2.搭建mysql数据库，创建语句文件doc/database/italkserver.sql
    3.运行
         第一种：Tomcat
         第二种: Jetty
